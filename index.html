About DSGPay

DSGPay (Direct, Secure & Global) is a comprehensive cross-border payment application enabling businesses and individuals to send, receive, and exchange money globally. Built with Flutter for cross-platform mobile deployment (iOS/Android).

Architecture

GetX MVC pattern with provider-based API layer

Architecture Pattern

The application follows a modular MVC architecture using the GetX framework:

View (Widget)
➔
Controller (GetxController)
➔
Provider (API)
➔
Model (Data)
View Layer       → Flutter Widgets (GetView/StatefulWidget)
                     Observes state with Obx() reactive bindings

Controller Layer → GetxController classes
                     Business logic, form validation, state management
                     Uses .obs reactive variables

Provider Layer   → API providers (Dio HTTP calls)
                     Each feature has its own provider class
                     Centralized through ApiService

Model Layer      → Dart data classes
                     fromJson/toJson serialization
                     Equatable for comparison

Module Structure (each feature)

lib/modules/{feature}/
  ├── controller/
  │   └── {feature}_controller.dart    // Business logic & state
  ├── binding/
  │   └── {feature}_binding.dart       // Dependency injection
  ├── {feature}_screen.dart            // Main screen widget
  └── widgets/                         // Feature-specific widgets

Security

Multi-layered security with certificate pinning, encryption, and device protection

Certificate Pinning

SHA256 fingerprint pinning via http_certificate_pinning package. Certificate file: assets/lets-encrypt-r3.pem. Android also uses network security config with pinning.

Jailbreak / Root Detection

FreeRASP (Talsec) integration detects jailbroken/rooted devices, emulators, debug mode, and tampering attempts at app startup.

Encryption

RSA public key encryption for passwords and PINs before transmission. Base64 encoding for credentials. File: lib/utils/helper/encrypt_helper.dart

Biometric Authentication

local_auth package for fingerprint/Face ID. Toggleable in More > Security > On-Device Authentication.

Secure Storage

flutter_secure_storage for tokens, user data, and sensitive information. Encrypted at rest on device.

App Locking

secure_application (custom git) for app locking when backgrounded. Prevents screen capture/recording.

Error Monitoring
